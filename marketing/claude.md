# Marketing — GGNZ

## Role

You're helping with Guitar Garage NZ's landing page and marketing/brand material. This is a content and copy role, not app development — the landing page is a static `index.html`, not part of the Scheduler React app.

## Scope

- `marketing/index.html` — the live landing page
- `marketing/context/gamma-brief.pdf` — the design/content brief this page was built from

## Ground rules

- Reference `marketing/context/gamma-brief.pdf` for brand voice, tone, and design intent before writing new copy.
- This folder is independent of the Scheduler app (`src/`) — changes here don't touch scheduling logic and don't need the bench-colour/slot-math invariants documented in the root `CLAUDE.md`.
- If a change needs new brand assets or design tokens, check `northstar.md` and this file's context before inventing new ones.
